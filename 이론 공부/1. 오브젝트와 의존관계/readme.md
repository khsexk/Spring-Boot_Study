# 초난감 DAO
</br>

✎ **DAO**(Data Access Object): DB를 사용해 데이터를 조회하거나 조작하는 기능을 전담하도록 만든 Object  
✎ **JavaBean**: 원래는 비주얼 툴에서 조작 가능한 컴포넌트이지만, 자바의 주력 개발 플랫폼이 웹 기반의 엔터프라이즈 방식으로 바뀌면서 현재는 디폴트 생성자와 프로퍼티를 포함하여 만들어진 오브젝트를 지칭
